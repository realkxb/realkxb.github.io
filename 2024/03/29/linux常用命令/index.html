linux常用命令

2024-03-29
Word count: 2.6k   |   Reading time≈ 11 min

包含文件处理命令：ls、cd、pwd、touch、mkdir、cp、mv、rm、cat、more、head、tail、ln
权限管理命令：chmod、chown、chgrp、umask
文件搜索命令：which、find、locate、updatedb、grep
包含帮助命令：man、info、whatis
压缩解压命令：gzip、gunzip、tar、zip、unzip、bzip、bunzip
网络通信命令：write、wall、ping、ifconfig
系统关机命令：shutdown、reboot

文件处理命令

1、ls [-ald] [文件/目录]
  功能：查看目录文件
  -a 显示所有文件，包含隐藏文件
  -l 详细信息显示
  -d 查看目录属性
2、cd [目录]
  功能：切换目录
  示例：
    切换到根目录：cd /
    切换到上一级目录：cd ..
3、pwd
  功能：显示当前工作目录
4、touch [文件名]
  功能：创建文件
  示例：
    touch newfile
5、mkdir [目录]
  功能：创建目录
  示例：
    mkdir newdir
6、cp
  功能：复制文件或目录
  用法：
    cp -R [源文件或目录][目的目录]
7、mv
  功能：移动文件或者更改文件名
  示例：
    将file2移动到目录dir2：mv file2 dir2
    将file1改名为file2：mv file1 file2
8、rm
  功能：删除文件或目录
  用法：
    将file2删除：rm file2
    将dir目录删除：rm -r dir
  参数：
    -i：删除前逐一询问确认
    -f 即使原档案属性设为只读，亦直接删除，无需逐一确认
    -r：将目录下的文件和目录全部删除
9、cat
  功能：显示文件内容
  用法：
    cat [文件名]
10、more
  功能：分页显示文件内容
  用法：
    more [文件名]
     f/空格：下一页
     回车键：下一行
     q：退出
11、head
  功能：查看文件前几行
  用法：
    head -num [文件名]：显示文件前num行
12、tail
  功能：查看文件后几行
  用法：
    tail -num [文件名]：显示文件后num行
13、 ln
  功能：创建链接文件
  用法： ln [参数][源文件或目录][目标文件或目录]
    ln file1 file1.hard:硬链接，以文件副本的形式存在。但不占用实际空间。
    ln -s file1 file1.soft:软链接，以路径的形式存在。

权限管理命令

1、chmod
  功能：改变文件或目录权限
  用法： chmod [{ugo}{+-=}{rwx}][文件或目录]
  用法： chmod [777][文件或目录]

类别 用户类型 说明
u user 文件所有者
g group 文件所有者所在组
o others 所有其他用户
a all 所有用户
 模式表
操作 说明
+ 为指定用户增加权限
- 去除指定用户权限
= 设置指定用户的权限
r 读权限，对应数字4
w 写权限，对应数字2
x 执行权限，对应数字1

2、chown
  功能：改变文件或者目录的所有者
  用法：chown [用户][文件或者目录]
3、chgrp
  功能：改变文件或目录所属组
  用法：chgrp [用户组][文件或目录]
4、umask
  功能：显示、设置文件的缺省选项
  用法：
     umask [文件名]
     umask -S [文件名] 以rwx显示新建文件或目录默认权限

文件搜索命令

1、which
  功能：显示系统命令所在目录
  用法：which [命令名称]
2、find
  功能：查找文件或目录
  用法：find [搜索路径][匹配条件][动作]
  匹配条件中可最常用的部份：
  -name pattern：按文件名查找，支持使用通配符 * 和 ?
  -type type：按文件类型查找，可以是 f（普通文件）、d（目录）、l（符号链接）等
  -size [+-]size[cwbkMG]：按文件大小查找，支持使用 + 或 - 表示大于或小于指定大小，单位可以是 c（字节）、w（字数）、b（块数）、k（KB）、M（MB）或 G（GB）
  -mtime days：按修改时间查找，支持使用 + 或 - 表示在指定天数前或后，days 是一个整数表示天数
  -user username：按文件所有者查找
  -group groupname：按文件所属组查找
  动作: 可选的，用于对匹配到的文件执行操作，比如删除、复制等
  find 命令中用于时间的参数如下：
  -amin n：查找在 n 分钟内被访问过的文件
  -atime n：查找在 n*24 小时内被访问过的文件
  -cmin n：查找在n分钟内状态发生变化的文件
  -ctime n：查找在 n*24 小时内状态发生变化的文件
  -mmin n：查找在 n 分钟内被修改过的文件
  -mtime n：查找在 n*24 小时内被修改过的文件
 在这些参数中，n 可以是一个正数、负数或零。正数表示在指定的时&间内修改或访问过的文件，负数表示在指定的时间之前修改或访问过的文件，零表示在当前时间点上修改或访问过的文件。
3、locate
  功能：搜索文件或目录
  用法：locate [搜索关键字]
4、updatedb
  功能：建立整个系统目录文件的数据库
  用法：updatedb
5、grep
  功能：在文件中搜索字串匹配的行并输出
  用法：grep [指定字串][源文件]

帮助命令

1、man
      功能：获得帮助信息
      语法：man [命令或配置文件]
2、info
      功能：获得帮助信息
      用法：info [关键字]
3、whatis
      功能：获得索引的简短说明信息
      用法：whatis [关键字]

压缩解压命令

1、gizp
      功能：压缩文件（.gz）
      用法：gizp 选项 [文件]
2、gunzip
      功能：解压缩.gz的压缩文件
      用法：gunzip 选项 [压缩文件]
3、tar
      功能：打包目录(.tar.gz)
      用法：tar 选项 [目录]
                        -c 产生.tar打包文件
                        -v 显示详细信息
                        -f 指定压缩后的文件名
                        -z 打包同时压缩
      tar命令解压缩语法：
                        -x 解包.tar文件
                        -v 显示详细信息
                        -f 指定解压文件
                        -z 解压缩
4、zip
      功能：压缩文件或目录（.zip）
      用法：zip 选项[-r] [压缩后文件名称] [文件或目录]
                        -r 压缩目录
5、unzip
      功能：解压.zip的压缩文件
      用法：unzip [压缩文件]
6、bzip2
      功能：压缩文件（.bz2）
      用法：zip 选项 [文件]
                        -k 产生压缩文件后保留原文件
7、bunzip
      功能：解压缩
      用法：bunzip2 选项 [-k] [压缩文件]
                        -k 解压缩后保留原文件

网络通信命令

1、write
      功能：向另外一个用户发信息，以Ctrl+D作为结束
      用法：write <用户名>
2、wall
      功能：向所有用户广播信息
      用法：wall [message] [文件名]
3、ping
      功能：测试网络连通性
      用法：ping IP地址
4、ifconfig
      功能：查看网络设置信息
      用法：ifconfig 选项[-a] [网卡设备标识]
                        -a 显示所有网卡信息

系统关机命令

1、shutdown
      功能：关机
      用法：shutdown -h now
2、reboot
      功能：重启系统
      用法：reboot

shell应用技巧

1、命令补齐：命令补齐允许用户输入文件名起始的若干个字
母后，按键补齐文件名
2、命令历史：允许用户浏览先前输入的命令并重新调用它们，用history命令可以显示命令列表，按方向键↑和↓可查找以前执行过的命令
3、管道
将一个命令的输出传送给另一个命令，作为另一个命令的输入
使用方法：命令1|命令2|命令3……|命令n
4、命令连接符
；
用；间隔的各命令按顺序依次执行
&&
前后命令的执行存在逻辑与关系，只有&&前面的命令执行成功后，它后面的命令才被执行
||
前后命令的执行存在逻辑或关系，只有||前面的命令执行失败后，它后面的命令才被执行
5、命令替换符
命令替换：将一个命令的输出作为另一个命令的参数
格式为：命令1 `命令2`
